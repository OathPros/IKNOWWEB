# KNOWIT Static Website

This project is a complete multi-page static website for a York University KNOWIT-style initiative. It is built with plain HTML, CSS, and JavaScript and is designed to work directly on GitHub Pages with no backend.

## Files

- `index.html` - homepage and initiative overview.
- `resources.html` - searchable/filterable resource hub.
- `support-pathways.html` - step-by-step support pathways and scenario cards.
- `updates.html` - timeline and roadmap of initiative progress.
- `about.html` - initiative scope, principles, audiences, and feedback info.
- `css/style.css` - shared design system, layout, responsive styles, and components.
- `js/main.js` - mobile menu toggle, active nav highlighting, and resource filtering logic.

## Preview locally

1. Download or clone the repository.
2. Open `index.html` from the repository root in a web browser.
3. Use the navigation links to view all pages.

## Publish with GitHub Pages

1. Upload these files to a GitHub repository.
2. Open the repository **Settings**.
3. Open **Pages**.
4. Under Source, choose **Deploy from branch**.
5. Select the `main` branch and the `/ (root)` folder.
6. Save, then open the generated URL when deployment completes.

## Notes

This is a static site. It does not require a backend, package installation, build tools, or server runtime.
