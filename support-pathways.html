Support pathways

Choose the right route to help

1. Identify the issue

Define what is blocked: account, learning tools, software, connectivity, or security.

2. Check the quick guidance

Use concise steps first to solve common issues quickly.

3. Use the right support channel

Choose self-service, live support, or service request based on urgency.

4. Escalate when needed

Escalate with clear details when timelines, access, or risk require faster handling.

Common scenarios

I cannot log in

Start here: Verify Passport York username and password reset options.

When to escalate: Locked account persists after reset.

Useful details to include: Username, error message, time attempted, device.

I need help with Duo

Start here: Check device registration and backup methods.

When to escalate: New device cannot be enrolled.

Useful details to include: Device type, Duo prompt behavior, screenshots.

I cannot access eClass

Start here: Confirm enrolment sync and browser cache cleanup.

When to escalate: Course still missing after 24 hours.

Useful details to include: Course code, section, screenshot, deadline impact.

I need software

Start here: Check software catalogue and licensing eligibility.

When to escalate: Required software is unavailable for your program.

Useful details to include: Software name, course or research need, operating system.

I think something is a cybersecurity issue

Start here: Stop clicking links and report suspicious activity immediately.

When to escalate: Account compromise or data exposure is suspected.

Useful details to include: Sender info, email headers, timestamps, impacted files.
